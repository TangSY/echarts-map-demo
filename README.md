# echarts map

通过地图API获取实时最新的geojson数据，摆脱geojson文件数据过期的烦恼。
省市区县下钻。实现真正的下钻至县级，而不是只到市级。
亦可下载最新geojson数据。

体验地址：[https://hxkj.vip/demo/echartsMap/](https://hxkj.vip/demo/echartsMap/)
